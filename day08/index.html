Advent of Code

Day 07

Instructions

Read a list of strings. Each string is contained in double quotes and contains random characters and the following escaped characters: \", \\ and \xab (where ab is a hex number).

Part 1. number of total characters (minus whitespaces) in the list of strings minus number of characters in value of string (unescaped characters count as 1, double quotes around string count as 0)

Part 2. escape all the special characters in the input (minus whitespaces) and recalculate the difference
